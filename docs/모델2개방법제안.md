모델을 2개로 나눔

모델 1 (분류모델)
- 전체 데이터에서 P(위험도 > 0 | x) 를 예측
- 의미 : 이 포인트가 사고 지점들과 유사한 특성을 가지는가
- 확률인 P 출력
- p = p(위험도 > 0 | x)

모델 2 (회귀모델)
- 위험도 > 0 인 데이터만 넣어 학습
- 위험도 크기 예측
- 위험도인 r 출력
-  r = E(위험도 | 위험도 > 0, x)

이후 최종적으로 p * r = 최종위험도점수 계산

최종시각화는 
 위험도_actual = 0
  AND 최종위험도점수_percent > 50


## 구조
모델 1: classifier
target = 1 if 위험도 > 0 else 0
output = p

모델 2: regressor
train data = 위험도 > 0 only
target = log1p(위험도)
output = r

최종 점수:
risk_score = p * expm1(r)

백분율:
risk_score_percent = risk_score / max(risk_score) * 100

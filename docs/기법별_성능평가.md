# 기법별 성능 평가

## 기준 평가 지표
* 분류모델
1. PR-AUC
2. Recall@Top10%
3. Precision@Top10%

* 회귀모델
1. MAE
2. RMSE



## 평가할 기법들
1. 단일모델 
2. 모델 2개 
3. Oversampling - SMOTE + 모델 2개
4. Undersampling Random under Sampling (비율 1대1) + 모델 2개
5. Undersampling Tomek Links + 모델 2개
6. Undersampling CNN + 모델 2개
7. Undersampling OSS + 모델 2개 

- UnderSampling, Oversampling 에는 imbalanced learn 라이브러리 사용할것
- 모델 2개를 쓰는 기법들은 모델 2 회귀모델은 위험도 > 0 인 데이터만 넣어서 학습할것


## 평가설계
- notebooks/기법별성능비교 에 각 기법들이 적용된 ipynb 파일 생성
- 각 ipynb 파일에서 각 기법들 실행하고 시각화하여 결과를 보고 표에 작성

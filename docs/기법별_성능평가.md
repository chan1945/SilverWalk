# 기법별 성능 평가

## 기준 평가 지표
* 분류모델
1. PR-AUC
2. Recall@Top10%
3. Precision@Top10%

- 모델 2 는 회귀모델로 모든 기법에서 sampling 이 적용되지 않은 기존데이터에서 위험도 > 0 인 데이터만 넣어서 학습하므로 비교가 의미 없음


## 평가할 기법들
1. 모델 2개 
2. Oversampling - SMOTE + 모델 2개
3. Undersampling Random under Sampling (비율 1대1) + 모델 2개
4. Undersampling Tomek Links + 모델 2개
5. Undersampling CNN + 모델 2개
6. Undersampling OSS + 모델 2개 

- UnderSampling, Oversampling 에는 imbalanced learn 라이브러리 사용할것
- 모델 2개를 쓰는 기법들은 모델 2 회귀모델은 sampling 이 적용되지 않은 기존데이터에서 위험도 > 0 인 데이터만 넣어서 학습할것


## 평가설계
- notebooks/기법별성능비교 에 각 기법들이 적용된 ipynb 파일 생성
- 각 ipynb 파일에서 각 기법들 실행하고 시각화하여 결과를 보고 표에 작성

## 평가표

모델 1 에 대한 평가표

아래 값은 각 기법의 모델 1 test prediction 파일을 기준으로 계산했다.

| 사용한 기법 | PR-AUC | Recall@Top10% | Precision@Top10% | F1@Top10% |
| --- | ---: | ---: | ---: | ---: |
| 모델 2개 | 0.739790 | 0.409731 | 0.817160 | 0.545796 |
| Oversampling - SMOTE + 모델 2개 | 0.730828 | 0.401424 | 0.800592 | 0.534730 |
| Undersampling Random under Sampling (비율 1대1) + 모델 2개 | 0.681522 | 0.376650 | 0.751183 | 0.501729 |
| Undersampling Tomek Links + 모델 2개 | 0.738218 | 0.410770 | 0.819231 | 0.547179 |
| Undersampling CNN + 모델 2개 | 0.434699 | 0.261015 | 0.520562 | 0.347693 |
| Undersampling OSS + 모델 2개 | 0.740009 | 0.411215 | 0.820118 | 0.547772 |

현재 결과 기준으로는 OSS, Tomek Links, 모델 2개 baseline이 비슷하게 높고, CNN과 RandomUnderSampling은 상대적으로 낮다.
